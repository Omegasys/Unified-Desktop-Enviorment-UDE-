# Unified Desktop Environment (UDE)

The **Unified Desktop Environment (UDE)** is a customizable, time-based desktop environment built with simplicity and flexibility in mind. UDE allows users to customize their desktop experience to match their preferences with a focus on theme switching, taskbar configuration, and easy-to-use real-time configuration updates.

## Key Features
- **Customizable Themes**: Switch between light, neutral, and dark themes, or create your own.
- **Time-Based Theme Switching**: Automatically switch themes throughout the day based on your local time.
- **Taskbar and Desktop Customization**: Easily adjust taskbar color, position, icon placement, and more.
- **Real-Time Configuration Updates**: Changes to the `.config` files are applied in real-time without restarting the system.

## Features
- Change taskbar orientation (top, bottom, left, right)
- Set time-based themes (morning, afternoon, evening)
- Configure taskbar hiding behavior
- Set desktop backgrounds and icons
- Extensive configuration via `.conf` files

## Installation
Follow the instructions in [INSTALL.md](INSTALL.md) to install UDE on your system.

## Getting Started
To get started, configure your `ude-config.conf` file located in the `config/` directory. You can modify the theme settings, taskbar settings, and more. Once your configuration is complete, run `main.py` to launch the desktop environment.
